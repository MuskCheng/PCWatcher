# PCWatcher 项目设计文档

## 1. 项目概述

**项目名称**：PCWatcher（电脑守护者）

**项目类型**：Windows 桌面后台监控工具

**核心功能**：后台运行的系统监控工具，通过系统托盘提供监控和配置功能，支持 PushMe 推送告警通知

**目标用户**：需要实时监控服务器或工作站状态的用户

---

## 2. 技术栈

| 组件 | 技术选型 | 说明 |
|------|----------|------|
| 开发语言 | Python 3.x | 跨平台，生态丰富 |
| 系统托盘 | pystray | 轻量级托盘实现 |
| GUI | tkinter | Python 内置 GUI 库 |
| 系统监控 | psutil | 获取 CPU/内存/磁盘/网络信息 |
| 温度监控 | wmi + OpenHardwareMonitor | Windows 温度获取 |
| 配置存储 | JSON | 轻量级配置管理 |
| 通知推送 | PushMe API | 手机端消息推送 |

---

## 3. 功能清单

### 3.1 托盘功能
- [x] 启动时直接最小化到系统托盘，不显示主窗口
- [x] 托盘图标状态指示：
  - 绿色：所有指标正常
  - 黄色：部分指标接近阈值
  - 红色：存在指标超过阈值
- [x] 右键菜单项：
  - "当前状态" - 显示状态概览弹窗
  - "配置设置" - 打开配置窗口
  - "立即推送" - 手动推送当前状态到 PushMe
  - "退出" - 关闭程序

### 3.2 配置功能（首次启动强制引导）
- [x] **PushMe 配置**
  - push_key 输入框
  - "测试连接" 按钮
- [x] **CPU 监控配置**
  - CPU 使用率阈值（默认 80%）
  - CPU 温度阈值（默认 70°C），可选
- [x] **内存监控配置**
  - 内存使用率阈值（默认 85%）
- [x] **硬盘监控配置**
  - 各分区单独设置阈值（默认 90%）
  - 显示所有可用分区供选择
- [x] **网络监控配置**
  - 网卡下拉选择（列出所有可用网卡）
  - 上传速度阈值（默认 10 MB/s）
  - 下载速度阈值（默认 10 MB/s）
- [x] **监控间隔**：固定 30 秒

### 3.3 监控功能
| 监控项 | 采集内容 | 数据来源 |
|--------|----------|----------|
| CPU | 使用率 (%) | psutil.cpu_percent() |
| CPU | 温度 (°C) | OpenHardwareMonitor / WMI |
| 内存 | 已用/总量 (GB)、使用率 (%) | psutil.virtual_memory() |
| 硬盘 | 各分区 已用/总量/使用率 | psutil.disk_usage() |
| 网络 | 选定网卡 IP 地址 | psutil.net_if_addrs() |
| 网络 | 上传/下载速度 (B/s) | psutil.net_io_counters() |

### 3.4 通知功能
- [x] **PushMe 推送**
  - 使用 PushMe API 发送告警
  - 支持 markdown 格式消息
  - 消息包含：指标名称、当前值、阈值
- [x] **托盘气泡通知**
  - 使用 Windows 原生通知
  - 点击可打开配置窗口
- [x] **通知去重**
  - 同一指标告警后，5分钟内不重复通知

### 3.5 开机自启
- [x] 注册 Windows 启动项（HKCU\Software\Microsoft\Windows\CurrentVersion\Run）
- [x] 启动时使用 `--minimized` 参数，确保托盘运行

### 3.6 配置持久化
- [x] 配置文件路径：`%APPDATA%/PCWatcher/config.json`
- [x] 首次启动检测无配置文件，启动配置引导
- [x] 配置窗口可随时修改配置并保存

## 5. 模块设计

### 5.1 核心模块

| 模块 | 职责 |
|------|------|
| `pcwatcher.py` | 程序入口，处理命令行参数，创建托盘 |
| `config.py` | 配置管理：加载/保存 JSON 配置 |
| `monitor.py` | 监控引擎：采集各项指标 |
| `notifier.py` | 通知模块：PushMe 推送 + 托盘气泡 |
| `tray.py` | 托盘管理：图标、菜单、右键处理 |
| `gui.py` | 配置窗口：tkinter 界面 |

### 5.2 数据结构

**Config（配置）**:
```json
{
  "push_key": "xxx",
  "cpu_threshold": 80,
  "cpu_temp_threshold": 70,
  "memory_threshold": 85,
  "disk_thresholds": {
    "C:\\": 90,
    "D:\\": 90
  },
  "network_interface": "以太网",
  "network_upload_threshold": 10485760,
  "network_download_threshold": 10485760,
  "interval": 30
}
```

**MonitorData（监控数据）**:
```python
{
  "cpu_percent": 45.5,
  "cpu_temp": 55.0,
  "memory": {"used": 8.5, "total": 16.0, "percent": 53.1},
  "disks": [
    {"device": "C:\\", "used": 100, "total": 500, "percent": 20}
  ],
  "network": {
    "interface": "以太网",
    "ip": "192.168.1.100",
    "upload_speed": 1024,
    "download_speed": 2048
  }
}
```
